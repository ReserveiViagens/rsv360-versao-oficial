## 🎯 Visão Geral

O Sistema de Tickets permite que você solicite suporte, reporte problemas ou faça perguntas de forma organizada e eficiente.

### Benefícios

- ✅ **Organização:** Todos os seus pedidos em um só lugar
- ✅ **Rastreamento:** Acompanhe o status de cada ticket
- ✅ **Histórico:** Veja todo o histórico de comunicação
- ✅ **Notificações:** Receba atualizações em tempo real

## 📝 Tutorial de Criação de Ticket

### Passo 1: Acessar o Sistema de Tickets

1. Faça login na sua conta
2. Acesse o menu "Suporte" ou "Tickets"
3. Ou acesse diretamente: `https://seu-site.com/tickets`
4. Clique em "Novo Ticket"

**Tela inicial:**

```
┌─────────────────────────────────────┐
│  🎫 SISTEMA DE TICKETS              │
├─────────────────────────────────────┤
│                                     │
│  [Novo Ticket] [Meus Tickets]       │
│                                     │
│  Tickets Abertos: 2                 │
│  Tickets Resolvidos: 15             │
│                                     │
└─────────────────────────────────────┘
```

---

### Passo 2: Preencher Informações do Ticket

Você precisará informar:

1. **Assunto** (obrigatório)
   - Seja claro e objetivo
   - Exemplo: "Problema com check-in digital"

2. **Categoria** (obrigatório)
   - Escolha a categoria mais adequada:
     - 🐛 Bug/Problema Técnico
     - ❓ Dúvida/Pergunta
     - 💡 Sugestão
     - 🔐 Problema de Acesso
     - 💳 Problema de Pagamento
     - 📅 Problema com Reserva
     - 🏨 Problema com Propriedade
     - ⚙️ Outro

3. **Prioridade** (opcional)
   - Baixa: Não urgente
   - Média: Importante mas não urgente
   - Alta: Urgente
   - Crítica: Muito urgente (apenas para problemas críticos)

4. **Descrição** (obrigatório)
   - Descreva o problema ou dúvida em detalhes
   - Inclua passos para reproduzir (se for bug)
   - Adicione informações relevantes

**Exemplo de formulário:**

```
┌─────────────────────────────────────┐
│  Criar Novo Ticket                  │
├─────────────────────────────────────┤
│                                     │
│  Assunto: [________________]        │
│                                     │
│  Categoria: [▼ Selecionar]          │
│  • Bug/Problema Técnico             │
│  • Dúvida/Pergunta                  │
│  • Sugestão                         │
│  • ...                              │
│                                     │
│  Prioridade: [▼ Selecionar]         │
│  • Baixa                            │
│  • Média                            │
│  • Alta                             │
│                                     │
│  Descrição:                         │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │                             │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│  [Cancelar] [Criar Ticket]          │
└─────────────────────────────────────┘
```

**Dicas para uma boa descrição:**
- ✅ Seja específico e detalhado
- ✅ Inclua passos para reproduzir o problema
- ✅ Mencione quando o problema começou
- ✅ Informe o navegador/dispositivo usado (se relevante)
- ✅ Adicione screenshots se possível

---

### Passo 3: Adicionar Anexos (Opcional)

Você pode anexar arquivos ao ticket:

**Arquivos aceitos:**
- 📷 Imagens (JPG, PNG, GIF)
- 📄 Documentos (PDF, DOC, DOCX)
- 📹 Vídeos (MP4, AVI)
- 📊 Outros (ZIP, TXT)

**Como adicionar:**

1. Clique em "Anexar Arquivo"
2. Selecione o arquivo do seu computador
3. Aguarde o upload
4. Repita para múltiplos arquivos

**Exemplo:**

```
┌─────────────────────────────────────┐
│  Anexos                             │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────┐                    │
│  │   📎        │                    │
│  │ Anexar      │                    │
│  └─────────────┘                    │
│                                     │
│  Arquivos anexados:                 │
│  • screenshot.png (2.3 MB) [X]      │
│                                     │
└─────────────────────────────────────┘
```

**Dicas:**
- ✅ Screenshots ajudam muito a explicar problemas
- ✅ Limite o tamanho dos arquivos (máx. 10MB por arquivo)
- ✅ Use nomes descritivos para os arquivos

---

### Passo 4: Revisar e Enviar

Antes de enviar, revise:

1. ✅ Assunto está claro?
2. ✅ Categoria está correta?
3. ✅ Descrição está completa?
4. ✅ Anexos foram adicionados (se necessário)?

**Tela de revisão:**

```
┌─────────────────────────────────────┐
│  ✅ Revisar Ticket                   │
├─────────────────────────────────────┤
│                                     │
│  Assunto: Problema com check-in     │
│  Categoria: Bug/Problema Técnico    │
│  Prioridade: Alta                   │
│                                     │
│  Descrição:                         │
│  Não consigo fazer check-in...      │
│                                     │

│                                     │
```

**Ações:**
- ✅ Revise todas as informações
- ✅ Clique em "Enviar Ticket"

### Passo 5: Receber Confirmação

Após enviar, você receberá:

1. **Número do Ticket**
   - Use este número para referência
   - Exemplo: #TKT-12345

2. **Email de Confirmação**
   - Com todos os detalhes do ticket
   - Link para acompanhar o ticket

3. **Notificações**
   - Você receberá notificações quando houver atualizações

**Tela de confirmação:**

Ao clicar em um ticket, você verá:

1. **Informações do Ticket**

## ❓ FAQ

### Quanto tempo leva para receber uma resposta?

**R:** Depende da prioridade:
- **Crítica:** Até 1 hora
- **Alta:** Até 4 horas
- **Média:** Até 24 horas
- **Baixa:** Até 48 horas

### Posso editar um ticket após criar?

**R:** Você pode adicionar comentários, mas não pode editar o ticket original. Se precisar corrigir algo, adicione um comentário explicando.

### Como fechar um ticket?

**R:** Se o problema foi resolvido, você pode fechar o ticket clicando em "Marcar como Resolvido" nos detalhes do ticket.

### Posso reabrir um ticket fechado?

**R:** Sim! Se o problema voltar ou não foi totalmente resolvido, você pode reabrir o ticket.

### E se eu não receber resposta?

**R:** Verifique sua caixa de spam. Se não encontrar, entre em contato pelo chat ou telefone informando o número do ticket.

### Posso criar múltiplos tickets?

**R:** Sim, mas recomendamos criar um ticket por problema para facilitar o acompanhamento.

### O que fazer se o problema for urgente?

**R:** Marque a prioridade como "Alta" ou "Crítica" e entre em contato também pelo chat ou telefone.

### Posso anexar vídeos?

**R:** Sim! Você pode anexar vídeos de até 10MB. Para vídeos maiores, use um serviço de compartilhamento e envie o link.
