# Variáveis de Ambiente Adicionais

Adicione estas variáveis ao seu arquivo `.env` no backend:

```env
# Google Hotel Ads
GOOGLE_HOTEL_CENTER_API_KEY=
GOOGLE_ADS_API_KEY=

# Twilio Voice Commerce
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE_NUMBER=
TWILIO_WEBHOOK_BASE_URL=http://localhost:5000

# OpenAI Voice Commerce
OPENAI_API_KEY=
```

## Instruções de Configuração

1. **Twilio**: Obtenha credenciais em https://www.twilio.com/
2. **OpenAI**: Obtenha API key em https://platform.openai.com/
3. **Google Hotel Ads**: Configure no Google Ads Center
