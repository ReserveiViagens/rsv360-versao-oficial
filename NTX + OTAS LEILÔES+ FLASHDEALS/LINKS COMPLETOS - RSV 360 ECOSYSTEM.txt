## 20. NOTAS IMPORTANTES

- A maioria das páginas requer autenticação
- Use `/login` para acessar páginas protegidas
- Tokens JWT são necessários para APIs

### 20.2 Parâmetros Dinâmicos
- `[id]` = ID do recurso (ex: booking ID, customer ID)
- `:id` = Parâmetro de rota (ex: `/bookings/:id`)
- Substitua pelos valores reais ao acessar

- **Produção:** `http://www.reserveiviagens.com.br`
