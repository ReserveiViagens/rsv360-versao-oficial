## 🎨 DESIGN SYSTEM ADOTADO

### Cores Padronizadas
```
Primary:     #0066CC (Azul vibrante) - Links, Botões, Seleções
Secondary:   #00AA66 (Verde) - Promoções, Sucesso, PIX OFF
Gray 50:     #F9F9F9 (Backgrounds claros)
Gray 900:    #1A1A1A (Textos principais)
Success:     #22C55E (Verde sucesso)
Warning:     #F59E0B (Laranja alerta)
Error:       #EF4444 (Vermelho erro)
```

### Tipografia
```
Display:     Poppins Bold (36px) - Títulos principais
Heading:     Poppins SemiBold (24px) - Seções
Subheading:  Poppins Medium (18px) - Subtítulos
Body:        Inter Regular (16px) - Conteúdo
Small:       Inter Regular (14px) - Detalhes
```

Button        - Primary, Secondary, Outline
